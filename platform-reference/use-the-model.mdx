Congratulations! You now have a fine-tuned, custom Synapse AI model.

### Get Your Integration Code

You can get your integration code by clicking on the **"Code Snippet"** button to start using your custom model in your applications.

![Code Snippet Interface](../images/hero-dark.png)

### Model Management

You can rename your model by clicking on **settings** next to the Code Snippet button. You can also share it with your team members for collaborative development.

### Need to Retrain?

If you find that you might need to retrain it, go back to the **upload data** page or the **fine-tune** page.
